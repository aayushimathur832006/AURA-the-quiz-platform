# AURA // ENGINE

AURA Engine is a futuristic, terminal-styled assessment and sandbox environment designed to validate computational logic, run tech quizzes, and track performance metrics.

## 🚀 Features

- **Secure Access Terminal:** High-tech, minimal interface for operator authentication.
- **Academic & Production Sandboxes:** Choose from multiple technology stacks and question capacities to test your skills.
- **Telemetry & Ledger:** Real-time diagnostic registry to track your scores, validation accuracy, and time taken.

## 🛠️ Tech Stack

- **Frontend:** React (Vite), Glassmorphism UI, LocalStorage
- **Backend:** Node.js, Express
- **Database:** MongoDB

## 📦 Local Setup

1. Clone the repo & navigate to the project directory.
2. Install backend dependencies and start the server (`node server.js`).
3. Install frontend dependencies and start the dev server (`npm run dev`).
4. Make sure your local MongoDB instance/cluster is running.
